# Natarajan Muthu AI Portfolio Website

This is a static AI-focused portfolio website ready for GitHub Pages.

## Free Hosting Recommendation

Use GitHub Pages with the repository name:

```text
natarajan0007.github.io
```

After publishing, the free website URL will be:

```text
https://natarajan0007.github.io
```

## Deploy To GitHub Pages

1. Create a new public GitHub repository named `natarajan0007.github.io`.
2. Upload the contents of this `resume-site` folder to the root of that repository.
3. Commit the files to the `main` branch.
4. Open `Settings > Pages` in GitHub.
5. Choose `Deploy from a branch`, select `main`, and use `/root`.
6. Wait a minute or two, then open `https://natarajan0007.github.io`.

## Files

- `index.html` - the AI portfolio website.
- `styles.css` - responsive styling.
- `script.js` - terminal typing, canvas animation, counters, copy email and active navigation.
